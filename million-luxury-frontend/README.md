Million Luxury exercise.

## Como usar?

1. Instala las dependencias
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

2. Ejecuta el proyecto

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) En tu navegador.
